<h2>Sobre o Projeto:</h2>
Este projeto foi criado a partir de uma necessidade pessoal durante estudos de desenho:
encontrar referências visuais rapidamente, sem distrações.

<h2>Com apenas um clique, a aplicação gera uma nova imagem aleatória, ideal para:</h2>
<ul>
  <li>treino de observação</li>
  <li>estudo de proporção, luz e composição</li>
  <li>prática de desenho sem apego à cópia</li>
</ul>

<h2>Tecnologias Utilizadas:</h2>
<ul>
  <li>React</li>
  <li>Vite</li>
  <li>Chakra UI (v3)</li>
  <li>Axios</li>
  <li>TypeScript</li>
  <li>Vercel (deploy)</li>
</ul>

<h2>Pré-requisitos:</h2>
<h3>Antes de rodar o projeto, certifique-se de ter instalado:</h3>
<ul>
  <li>Node.js ≥ 20</li>
  <li>npm</li>
</ul>

<h2>Como Rodar Localmente:</h2>
<ol>
  <li>git clone https://github.com/NicklausGabriel/Refer-ncia-Aleat-ria-frontend.git</li>
  <li>cd React-Vite-bike-shop/</li>
  <li>npm install</li>
  <li>npm run dev</li>
</ol>
